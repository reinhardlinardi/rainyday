==== RainyDay =====

System Requirement :
1. Windows 10
2. Visual Studio 2017

How to Run :
1. Open .sln file in Visual Studio
2. Run project, wait until your browser opens RainyDay
3. Click Search in navigation bar
4. Type keyword and select string matching algorithm, the result will be displayed in real time
5. If you want to update news, click refresh button and wait until RainyDay tells you "RSS is updated".
